Support forum

There is a mailing list available for discussions and support for Graph. A forum and newsgroup is also available as alternative interfaces to the list. It is the same messages with three different interfaces.

Mailing list
When subscriped to the Graph mailing list you receive and email each time a message is posted to the list. Subscription may be done from the info page.

To post a message to the list, send an email to graph-support@lists.sourceforge.net

You can find a collection of all prior messages in the Archives.

Forum
Instead of using the mailing list directly you can read it and write to the list as a forum at Gmane:
http://news.gmane.org/gmane.comp.mathematics.graph

To post messages you need a working email address. The first time you post a message you will receive an email you need to reply to, to confirm that you are a real person. To prevent spam your email address will be shown a little obfuscated in the forum message. More information may be found in the Gmane FAQ.

Newsgroup
Gmane also supply a newsgroup interface to the list, which enables you to use your favorite newsgroup client to interface to the list:
nntp://news.gmane.org/gmane.comp.mathematics.graph

The above link should subscripe your default newsgroup client to the graph newsgroup. If not you have to check the documentation following your client for how to subscripe to it.
