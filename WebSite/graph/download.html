Download

Click on the link below to download the Graph. The program is open source (GNU GPL) and you may give it to whomever you like.

SetupGraph-4.2.exe (2.8 MB)

 

There is also a limited edition for schools that do not allow abstract algebraical symbolic manipulation. It is the same as the normal version except that differentiation is not supported.

SetupGraphLimited-4.2.exe (2.8 MB)

 

Both the binary programs and the source code is available from SourceForge.

Installation

The file is a self extracting installation program. You just execute the file and the installation will ask in which directory you want to install Graph. The installation will add a shortcut to Start->Programs, which may be used to start Graph.

If an older version of Graph is already installed, you can just install the new version over it. There is no need to uninstall the old version first, but make sure Graph is not running while installing.

If you later wish to remove the program, you just use Add/Remove Programs from the Control Panel.

Under the installation you can choose between different languages for the user interface. This may later be changed from inside the program.
